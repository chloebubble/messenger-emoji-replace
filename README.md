# Messenger System Reactions

Userscript that replaces Facebook-style Messenger reaction icons with system emoji glyphs.

## Supported URLs

- `https://www.messenger.com/*`
- `https://www.facebook.com/messages/*`

## Install

Install [`messenger-system-reactions.user.js`](/Users/chloe/devstuff/messenger-emoji-replace/messenger-system-reactions.user.js) in your userscript manager, then open either supported Messenger web app URL.
